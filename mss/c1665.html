Menstrua   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 6.

Description:   A note in Locke’s hand, with the marginal caption “Menstrua”, and intialed by Locke. The note is undated, but was written ca. 1665.

Publications:   None

Pestis L   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 8.

Description:   Several notes in Locke’s hand, with the marginal caption “Pestis L”, and intialed by Locke. The notes are undated, but was written ca. 1665, although they may not all be of the same date.

Publications:   None

Pleuritis   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 9.

Description:   A note in Locke’s hand, with the marginal caption “Pleuritis”, and intialed by Locke. The note is undated, but was written ca. 1665; it is the first (and shortest) note on this topic on the same page.

Publications:   None

Pleuritis   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 9.

Description:   A note in Locke’s hand, with the marginal caption “Pleuritis”, and intialed by Locke. The note is undated, but was written ca. 1665; it is the second (and longest) note on this topic on the same page.

Publications:   None

Febris annua   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 12.

Description:   A note in Locke’s hand, with the marginal caption “Febris annua”, and intialed by Locke. The note is undated, but was written ca. 1665.

Publications:   None

Febris   [ca. 1665]

Location:   Bodleian Library, MS. Locke d. 9, p. 13.

Description:   A note in Locke’s hand, with the marginal caption “Febris”, and intialed by Locke. The note is undated, but was written ca. 1665; it is later than earlier notes on the same page.

Publications:   None
